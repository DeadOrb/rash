# RASH Validator

## Requirements

- [node.js](https://nodejs.org/en/)
- [web.py](http://webpy.org/)
- [lxml](http://lxml.de/installation.html)
- [pyjing](https://pypi.python.org/pypi/jingtrang)

## Usage

Start the client:
```sh
cd client
npm start
```

Start the server:
```
cd server
python webapp.py
```
