Abstract

It is possible to specify some special sections if their headings contains specific words. In particular, if the first section of the document is named Abstract, then it actually represents the abstract of the paper.

Similarly, if the last section of the document is named References and contains an ordered list, then it actually represents the bibliography of the paper.

Common section

This is just a common section as shown in previous testbeds.

Acknowledgements

Another special section is the Acknowledgements one, which is used to thank people involved (to some extent) in the work. Usually, this section is placed before the reference section.
