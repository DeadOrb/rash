Formulas

In this example we want to introduce how to use formulas within the text.

Inline formula

Inline formulas are created using the formula editor of OpenOffice: a − b = ¬ c a - b = neg c .

Formula box

Formula boxes are created by simply including them in a new paragraph with a caption and without any other textual element in it, as follows.

a − b = ¬ c a - b = neg c
